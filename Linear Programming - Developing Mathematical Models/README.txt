# Linear Programming - Developing Models

This project explores the development of linear programming models for various optimization problems.

## Contents
- Introduction to linear programming
- Model formulation
- Example problems and solutions

## Usage
1. Clone the repository
2. Navigate to the `Linear_Programming_Developing_Models` folder
3. Open the relevant files or `notebook.ipynb`
4. Follow the instructions to see the analysis

## Requirements
- Python 3.x
- Jupyter Notebook
- Required Python libraries: numpy, scipy, matplotlib, etc.

# Integer and Mixed-Integer Programming

This project focuses on optimization techniques for problems involving integer and mixed-integer variables.

## Contents
- Introduction to integer programming
- Mixed-integer programming techniques
- Example problems and solutions

## Usage
1. Clone the repository
2. Navigate to the `Integer_and_Mixed_Integer_Programming` folder
3. Open the relevant files or `notebook.ipynb`
4. Follow the instructions to see the analysis

## Requirements
- Python 3.x
- Jupyter Notebook
- Required Python libraries: numpy, scipy, matplotlib, etc.
